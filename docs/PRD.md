# Save It - PRD (Product Requirements Document)

## 1. 제품 개요

### 1.1 제품 목적
사용자가 인터넷 사용 중 저장한 링크를 다시 확인하고 소비할 수 있도록 돕는 서비스.
기존 북마크는 "저장"에 초점이 맞춰져 있지만, 본 제품은 "재사용(다시 보기)"에 초점을 둔다.

### 1.2 핵심 가치
- 저장된 링크를 잊지 않게 한다
- 링크를 상황에 맞게 다시 노출한다
- 정보를 행동 중심으로 정리한다

> "저장만 되고 잊혀지는 링크를, 다시 보게 만드는 서비스"

## 2. 문제 정의

### 2.1 현재 사용자 문제
1. **링크 과다로 인한 탐색 실패** - 수십 개의 북마크 중 원하는 링크를 찾지 못함
2. **저장 후 망각** - "나중에 보려고 저장" → 실제로는 다시 보지 않음
3. **중요도 구분 불가** - 중요한 자료와 단순 참고 링크가 혼재
4. **관리 비용 증가** - 정리하려면 시간과 노력이 필요 → 방치

## 3. 타겟 사용자
- 자료 조사를 하는 대학생
- 업무상 링크를 많이 저장하는 직장인
- 유튜브/블로그 콘텐츠 소비자

## 4. 솔루션: PARA 기반 구조
| 분류 | 설명 | 예시 |
|------|------|------|
| Projects | 현재 진행 중 작업 | 과제, 업무 |
| Areas | 지속적 관심 분야 | 개발, 운동 |
| Resources | 참고 자료 | 나중에 볼 영상 |
| Archives | 완료/비활성 | 끝난 프로젝트 |

## 5. 핵심 기능

### MVP
1. 링크 저장
2. PARA 분류
3. 링크 재노출 (미열람, 최근 저장, 중요도 기반)

### 권장 기능
- 중요도 설정
- 태그 및 카테고리
- 자동 분류 추천

### 선택 기능
- 링크 수정/삭제/이동
- 상태 흐름 자동화 (Project → Archive)
- 리마인드/추천

## 6. 성공 지표 (KPI)
- 저장된 링크 재열람률
- 저장 후 7일 내 클릭 비율
- 링크 소비 완료율
- 사용자 재방문율
